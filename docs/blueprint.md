# **App Name**: Portfolio Hero

## Core Features:

- Hero Content: Display name, title, and a brief introduction.
- Visual Background: Include a visually appealing background (image or gradient).
- Call to Actions: Add links to social media profiles and a contact button.
- AI-Powered Dynamic Text: Use a generative AI tool to dynamically adjust the hero section's text based on keywords extracted from the user's latest projects. This adds a dynamic, SEO-friendly element to the portfolio.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) to convey trust and stability.
- Background color: Very light blue (#F0F4FF) provides a soft and clean backdrop.
- Accent color: Analogous cyan (#00BCD4) highlights key elements like links and buttons.
- Font Pairing: 'Space Grotesk' (sans-serif) for headlines and short blocks of text, paired with 'Inter' (sans-serif) for longer body text. 'Space Grotesk' has a modern feel while 'Inter' offers excellent readability.
- Use simple, outlined icons for social media links and contact button to maintain a clean and professional look.
- Employ a clean, grid-based layout with ample spacing for readability and visual appeal. Content should be centered or left-aligned depending on the visual balance.
- Subtle animations for hover effects on links and buttons, as well as a smooth transition when the AI updates the text content, to enhance user engagement without being distracting.
