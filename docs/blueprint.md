# **App Name**: MirrorFolio

## Core Features:

- Hero: Hero Section: Display a brief introduction with a professional photo and headline.
- About: About Section: Showcase personal background, skills, and mission with a compelling narrative.
- Experience: Experience Section: List professional history, projects, and achievements in a structured format.
- Theming: Visual Consistency: Ensure font usage, color scheme, and centered layout replicate the screenshot accurately.

## Style Guidelines:

- Primary color: Dark blue (#2E3148) to set a professional and trustworthy tone.
- Background color: Very light gray (#F0F0F4) to provide a clean and unobtrusive backdrop.
- Accent color: Muted purple (#807BA0) for highlights and interactive elements, ensuring they stand out gently without overwhelming the overall design.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern look, used consistently across all sections.  Note: currently only Google Fonts are supported.
- Centered Layout: Use a centered layout for all sections, providing balance and focus on content. Adapt margins based on screen size.
- Subtle Transitions: Use subtle, non-intrusive transitions (e.g., fade-in on scroll) to improve UX without being distracting.
